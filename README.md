# Пума
_**Пума**_ - четвёртый по величине представитель семейства кошачьих в мире и второй в Америке; крупнее неё только тигр, лев и ягуар. 

### Размеры:
- достигает в длину 100—180 см; 
- длина хвоста 60—75 см; 
- высота в холке 60—90 см;
- массе до 105 кг.
